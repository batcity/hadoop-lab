# Frequently asked Questions:


## What is Hadoop:

Hadoop is a framework that allows for distributed processing of large datasets across a cluster of machines.

## What are its core modules:

- Hadoop Common -> This contains the common utilities that other hadoop modules rely on
- YARN - Yet another resource Negotiator - this is the module responsible for job scheduling and managing resources on the cluster
- Hadoop Mapreduce -> This is a YARN based distributed processing engine designed to process large datasets in parallel
- HDFS - Hadoop Distributed file system - A distributed file system designed to store large data sets reliably across multiple machines
